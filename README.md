# Cloud Drive - Google Drive Clone

Full-stack cloud storage application with:
- React web client
- C# WPF desktop client with unit tests
- Node.js / Express backend
- File synchronization via polling


## Features

- Upload, download, and organize files
- Synchronization between web and desktop clients
- Authentication via Google
- Drag-and-drop file upload (web client only)
- Unit tests for desktop client
- Polling-based sync mechanism

## Getting Started

### Server
1. `cd server`
2. `npm install`
3. `npm start`

### Web Client
1. `cd web`
2. `npm install`
3. `npm start`

### Desktop Client
1. Open `desktop/CloudDrive.sln` in Visual Studio
2. Build and run
